## three-js-blackhole

### Features

- Drag to look around
- Adjustable quality vs performance
- Adjustable bloom effect
- Save to an image
- Physics
  - Lorentz transform
  - Doppler shift
  - Relativistic beaming
  - Accretion disk
  - Background star color from blackbody spectrum (random and trivial velocity for red shift)

1) npm install
2) npm run dev
